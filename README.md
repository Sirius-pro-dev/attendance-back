# Система контроля посещаемости занятий

Система, которая позволит студентам и преподавателям отслеживать и контролировать посещаемость занятий.

## Цель

Создать безопасную систему для отслеживания посещаемости занятий студентами и преподавателями.

## MVP

- Возможность входа пользователя в систему(регистрация и аутентификация)
- Создание встречи
- Отметка присутствия
- Генерирование QR-кода
- Функционал администратора

## Стек технологий Backend

- [fastify](https://fastify.dev/)
- [mongodb](https://www.mongodb.com/)
- [mongoosejs](https://mongoosejs.com/)
